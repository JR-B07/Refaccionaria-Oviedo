PROBLEMA: MySQL no inicia
SOLUCION: 
  1. Ejecutar como Administrador
  2. Verificar: net start MySQL
  3. O revisar logs: server.log

PROBLEMA: Puerto 8001 en uso
SOLUCION:
  1. Encontrar proceso: netstat -ano | find "8001"
  2. Matar proceso: taskkill /PID xxxx /F
  3. O cambiar puerto en .env

PROBLEMA: Dependencias faltantes
SOLUCION:
  1. Ejecutar: pip install -r requirements.txt
  2. O ejecutar cualquier launcher que las instala

PROBLEMA: Base de datos vacia
SOLUCION:
  1. Ejecutar: python scripts/inicializar_datos.py
  2. O ejecutar Refaccionaria.bat que lo hace automaticamente
